# cheetah for PAL-BL11C
1. Install the requirement libraries.
2. Modify 'compileme' and execute 'compileme'
3. cd build
4. make
5. make install


## Requirements
- cmake==3.19.2
- boost==1.53.0
- HDF5==1.8.21
- CBFlib==0.9.5.1
