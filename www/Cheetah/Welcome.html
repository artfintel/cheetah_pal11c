Welcome to Cheetah

 
 

Usage examples include:

  1. Hit finding

  2. Serial crystallography (take home only events containing crystal diffraction)

  3. Single particles (take home only particle hits)

  4. Sorting of data

  5. Sort pump-probe data into laser-on and laser-off states

  6. Sorting by event code or other criteria

  7. Data reduction

  8. Radial stacks for use in SAXS/WAXS analysis (radial line out for each event, sorted)

  9. Per pixel histograms of the entire detector (for detector calibration and characterisation)

  10. Peak lists (location of all found peaks)

  11. Compilation of powder patterns (sums of data in a run, sorted)

  12. Diagnostics synchronised by event for subsequent sorting

  13. Timing tool for Femtosecond pump-probe timing (trace of femtosecond timetool matched with events)

  14. Per-shot photon spectrum

  15. XES spectrometers and TOFs

  16. Detector correction

  17. Output from some detectors is simpler than others

  18. Includes: dark and gain corrections, cspad common mode corrections, pnCCD corrections

  19. Conversion from ADU to photon counts

  20. Output of data in facility-independent HDF5 files readable by 3rd party programs

  21. Detector frames of interest saved in HDF5 format (standard .cxi files)

  22. Diagnostic data in HDF5 format synchronised by event

  23. Conversion of selected events (for example when developing algorithms offline)

  24. Consistent format shields you from the multiple file formats and APIs used at different facilities


Cheetah itself consists of a GUI front end (Python+Qt) to coordinate processing and view output, which calls backend programs that can also be called from the command line if desired.  Output is in the form of portable, standardised HDF5 and plain text files.  Code is modular and can easily be adapted to any source of serial imaging data, including data collected using both free electron laser and synchrotron sources using a variety of detectors (including csPAD, pnCCD, AGIPD, Pilatus, Rayonix).  Cheetah is open source and released under the GNU GPL v3 license.  Re-use of core algorithms such as peak finders is encouraged and the code is written to simplify re-use as far as practical.

Please cite this paper if you have used Cheetah or a part of Cheetah in your data analysis.

doi:10.1107/S1600576714007626 - Download PDF - Article on IUCr website

Cheetah is a set of programs for processing serial diffraction data from free electron laser sources, and which enable taking home only the data with meaningful content.  Developed for use in our own experiments this is a sanity saver in many serial imaging experiments.
