File formats

Broadly speaking: either HDF5 or plain text for both image and supporting data.

Event data, images and results

The current format is described here:

    Current format:  HDF5 file for results and metadata, CXI file for image data



Earlier file formats are also documented:

    2014-2017 format: Single large .cxi files (many images per file)

    2009-2014 format: Small single-frame HDF5 files (one file per image)



Additional data (powder patterns, mask files)

    Saved in HDF5 format as listed below
